# Bot Narrador - Operação Raghnild

Este é um bot interativo de Telegram com narrativa gamificada para a missão Operação Raghnild.

### Comandos:

- /start
- /introducao
- /cofre1
- /cofre2
- /salavip1
- /salavip2
- /fuga

### Sistema de rolagem:

- /roll_outfit
- /roll_camorra
- /roll_famiglia

Desenvolvido em Flask. Pronto para deploy no Render
